# 🐺 Project 4 Starter Sandbox: Multi-Agent Micro-Market Game

这是你专属的 Project 4 沙盒。这是全课的最巅峰试炼。你将脱离玩家身份，成为这片黑暗森林的上帝。

### 🚨 前序环境搬运
本仓库是空架构。**请从你的 P1/P2 中把 `simulator.py` (V3 底层机器) 搬运到本项目的 `src/` 目录下**。它是全世界所有智能体的斗兽场。我们要摧毁它，然后再拯救它。

### 📂 目录导航
- `src/master_model.py`：继承了 `mesa.Model` 的全局引擎，你必须在这里掌管主大钟（Tick）的数据刻录。
- `src/agents/`：包含了三个不同智能体（噪音散户、有毒套利者、主动内卷 LP）的基础 `mesa.Agent` 代码槽位。
- `tests/test_milestone_1.py`：跑通你全流程架构（确保所有 Agent 都能顺利发出 Action，引擎不出 bug）的基础极压验证。
- `experiments/`（请自行创建）：写你的最终 DAO Proposal 机制回测脚本。
